補正係数

■係数について
なにこれ
「技性能評価」は全ての技の性能を1つの数値で評価する試みです。
今回はバトルの状況を「牽制」「優勢維持」「劣勢逆転」「超必殺」の4つに分け、状況別の技性能評価を試みています。
バトルの状況
牽制
　序盤に使いやすい技です。与えるダメージが多く、命中率が高く、消費ガッツの少ない技を高めに評価します。
　自爆ダメージが大きいリスクの大きい技の評価は高くならないようにしています。

優勢維持
　序盤から終盤までライフで勝っているときに使いやすい技です。与えるガッツダウン、命中率、消費ガッツの少ない技を高く評価します。
　自爆ダメージは優勢を自ら手放しかねないので評価は高くならないようにしました。

劣勢逆転
　序盤から中盤までライフで負けているときに使いやすい技です。与えるダメージが多い技を高く評価します。
　劣勢下なので命中率やコスト、自爆ダメージなど多少のリスクはある程度許容するようにしています。

超必殺
　終盤まで維持してきたガッツを消費して放つ大技です。とにかくダメージを与えやすい技を最大限に評価します。
　勝敗確定まで数秒の状況を想定しており、消費ガッツは度外視です。
計算式
私感が一番入っている箇所です。ここが一番信用できません。

評価値 = ( ライフダメージ + ガッツダメージ + 状態異常 ) * 命中 * ガッツコスト * 距離補正
ライフダメージ = (ダメージ * ( 1 - ( クリティカル + 10 ) / 100 ) + ライフダメージ * 1.5 * ( クリティカル + 10 ) / 100
　　　　　　　　 + (ダメージ * ドレインダメージ / 100 ) * ドレイン係数 + 回復 * 回復係数 - 自爆ダメージ * 自爆係数 ) * ライフダメージ係数
　　　　　　　　ガッツダメージ = ガッツダメージ * ガッツダメージ係数 + (ガッツドレイン * /100) * ガッツドレイン係数
状態異常 = (状態異常率 / 100 * 状態異常係数)
命中 = ( 50 + 命中率 ) / 100 * 命中係数
ガッツコスト = 1 + 99/消費ガッツ * 消費ガッツ係数
距離補正 = 技がある距離の係数
評価値が0の技があるようだが？
自己強化、相手弱体化の特殊技です。
その技単体での評価が難しく、計算結果では0となりますが、使うだけ無駄な技というわけではありません。
そんな細かい立ち回り、公式大会のオートバトルじゃ難しいのでは？
はい。

精度は高くないと思います。どんどん変えましょう。

足りない評価項目
ヒット時、ミス時の経過時間
　これの有無でセブンガーの硬芯鉄拳弾なんかは超必殺評価値がだいぶプラスに変わると思います。
　しかし、全技の経過時間データが手元に存在しないため今回は評価対象外としています。

技使用後の移動
　技を使用すると使用後に微量の前方移動、後方移動が発生します。位置の変化なしの場合もあります。
　前方、後方への移動は他の技との連携が可能になり、位置変化がない場合は同じ技の連続使用が可能になります。
　このデータも手元に存在しないため今回は評価対象外です。
